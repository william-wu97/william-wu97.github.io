技嘉System Information Viewer安装不了的解决办法

发布时间 : 2020-04-23 最后修改 : 2021-04-24
上一篇 : Dell Inspiron 15-7557 10.15.3 EFI分享 下一篇 : Dell Inspiron 15-7557 10.15.5 EFI分享

最近在使用技嘉的APP Center时遇到一个问题，里面的SIV(System Information Viewer，如下图红框所示)卸载过后，怎么都重装不上，估计是注册表没删干净

# 尝试过的办法

  1. Win+R输入regedit打开注册表编辑器查找，找了一圈都没找到相关的注册表信息，无奈放弃
  2. 百度相关问题，在技嘉吧找到了一个帖子，为什么我的System Information Viewer安装不了，内容如下所示：

可惜的是，这个帖子并没有给出解决办法

  1. 重装系统（我脑子进水了才会为这破软件重装系统）

# 以上方法尝试无果后，那就只能换个思路从这破软件的安装文件入手了

  1. 首先，我们打开技嘉官网下载这软件的单独安装文件，如下图所示：

2. 解压后得到以下文件，如下图所示：

3. 把setup.exe拖到ExeinfoPE（查壳工具）中查看，可以得出这个安装包是InstallShield打包的，所以理论上网上应该可以找到对应的解包工具

4. 百度过后，找到了一个相应的解包教程，请问用 InstallShield 制作的安装包如何解包？

# 具体解包步骤如下：

  1. 在 https://github.com/lifenjoiner/ISx 下载 ISx.exe ，将 Install Shield 安装包解开成 MSI 安装文件；
  2. 在 https://bioruebe.com/blog/dev/uniextract/下载 uniextract2(大名鼎鼎的 uniextract 网友再更新版)，并升级到最新版本，就能解开 MSI 安装包。
  3. 解包完成。

# 实践结果如下：

  1. 首先，我们将setup.exe直接拖到ISx.exe上面，可得到MSI安装文件目录setup_u，如下图红色箭头所示：

setup_u目录详细内容如下图所示：

  1. 然后我们双击打开 UniExtract.exe

软件运行界面如下图所示：

选择setup_u目录下的SIV.msi进行提取

# 这里不知道为什么第一次提取的目录只有一个文件，再提取一遍才得到正确的MSI解包内容

  1. program files文件夹即为MSI解包后的内容，我们先来看看里面都有什么

双击打开program files\GIGABYTE\SIV下的ThermalConsole.exe，出现了一个熟悉的界面：

OK，收工

# 解包内容下载地址：蓝奏云 密码:9i03

# 注意：

  1. 这个System Information Viewer不能脱离APP Center使用，也就是APP Center必须先安装才行
  2. 这样解包得到的System Information Viewer并不能出现在APP Center中，如下图所示：
